# Interface_Team2


### 프로젝트 진행 기간

2022.10.08 ~ 2022.10.25  

<br>

### 프로젝트 제목
Onculture  

<br>

### 주제
지역의 문화 생활(공연, 전시) 정보 제공 웹사이트

<br>

### 팀명
포맨

<br>

### 팀원
김승열, 서혁준, 유영일, 이준희

<br>

### 설명
* 삼성멀티캠퍼스 인터페이스 개발 프로젝트
* 각 지역의 공연, 전시 등의 문화생활 정보를 제공한다.
* 추가적으로, News API를 활용하여 세계 문화 뉴스 정보를 제공한다.

<br>
